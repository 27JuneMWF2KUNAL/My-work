Form

form element will helps to get client data and send it to the server

There are tow methods for sending data to the server

  • GET => is default, less secure because its use URL for sending data to the server with the help of ? => query string, limitation 2048 char, not supporting file uploading, we bookmark URL with data, faster then post thats why 90% of search engines are using get method, we cn get back data after refreshing/reloading our page
  • POST => secure -> not using url, no limitation, support file upload, we cant bookmark url with data
